**Sistema Inteligente de Gestión para Gimnasios**

Sistema web completo diseñado para automatizar la administración de gimnasios: socios, membresías, pagos, vencimientos y comunicaciones por WhatsApp.

---

## ✨ Características Principales

- **Gestión completa de clientes** (registro, planes, fechas de pago)
- **Cálculo automático de vencimientos** y cambio de estados
- **Automatizaciones con WhatsApp** (recordatorios, avisos de vencimiento)
- **Panel de estadísticas** y exportación de datos
- **Sistema anti-spam** con control de avisos
- **Dashboard intuitivo** para empleados y dueños

---

## 🛠️ Tecnologías Utilizadas

- **Backend**: PHP
- **Frontend**: HTML, CSS, JavaScript
- **Base de Datos**: MySQL
- **Automatizaciones**: n8n + Evolution API (WhatsApp)
- **Entorno**: XAMPP / VPS
